# Foundry

```
There are 4 things:
1. forge
2. cast
3. anvil
4. chisel

```


## Forge
### Commands

```
1. forge create $contractname --rpc-url $rpcurl --interactive  to deploy a contract
2. forge build to build the sol files
3. forge compile to compile the sol files 
4. forge script ./script/filename --rpc-url $rpculr --private-key $privatekey --broadcast to run the script 
5. forge fmt to format the sol files

```


## cast
## Commands

```
1. cast --to-base anyarbitarystring(hexadecimal) dec ---> converts hexadecimal to decimal 
2. cast send help ( to check the arguments to pass to send a transaction)
3. cast pass contractaddress "$functionsignature()" args --rpc-url $rpcurl --private-key $private_key ----> same for send as well.
```
